ファイアウォール

ネットワークを守る「門番」

ファイアウォールとは？

ファイアウォール（Firewall） とは、 ネットワークの出入口に置かれる 「門番」のような存在で、 通過する通信を監視し、 許可された通信だけを通し、 不正な通信や攻撃を遮断するための仕組みです。

インターネット と 社内ネットワーク の境界、 拠点間接続、 クラウド環境、 家庭用ルーター など、 さまざまな場所で使われています。

ファイアウォールは、 「通信を通すか・止めるか」 を判断するセキュリティ機能です。

どうやって通信を判断する？

ファイアウォールは、 管理者が設定した ポリシー（ルール） に基づいて通信を制御します。

・IPアドレス
送信元・宛先の住所情報

・ポート番号
80 / 443 / 22 など

・プロトコル
TCP / UDP など

・セッション状態
通信が確立済みかどうか

・アプリケーション
Web / DNS / SSH など

ファイアウォールの種類

種類 特徴
パケットフィルタ型 IPアドレスやポート番号だけで 通信を判断する方式
ステートフル型 通信状態（セッション）を追跡し、 正規通信か判断する方式
アプリケーション型 HTTPやDNSなど、 通信内容まで解析する方式
UTM / NGFW IPSやアンチウイルスなども 統合された多機能型

スループットとは？

ファイアウォールの性能は、 どれだけ通信を処理できるかを表す スループット で示されます。

・ファイアウォールスループット
基本通信の処理能力

・VPNスループット
VPN暗号化時の処理能力

・IPSスループット
IPS有効時の処理能力

セキュリティ機能を増やすほど、 CPU負荷が増えるため、 実際の速度は カタログ値より低下することがあります。

なぜ必要？

インターネットへ接続すると、 外部から不正アクセスを受ける危険があります。

そのため、 必要な通信だけを許可し、 危険な通信を止める セキュリティの壁 が必要になります。

現代では、 ファイアウォールは ほぼ必須のセキュリティ機能 です。

まとめ

ファイアウォールは、 ネットワークの出入口で通信を監視し、 ルールに基づいて通信を制御する仕組み です。

不正アクセスや攻撃から ネットワークを守るための、 非常に重要なセキュリティ技術です。
